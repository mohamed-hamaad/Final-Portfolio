# Personal Portfolio & Developer Brand

Welcome to the repository of my personal portfolio website. This project is a production-ready web application designed to showcase my software engineering skills, professional experience, and freelance web development projects. Built from scratch with a focus on modern design principles, performance, and clean architecture.

## ✨ Features
* **Modern CSS Grid Layout:** Completely custom responsive grid system crafted for seamless cross-device compatibility (Desktop, Tablet, Mobile) without relying on bulky UI frameworks.
* **Django 6 Backend:** Powered by the latest long-term stable Python web framework for secure, scalable routing and logic.
* **Production-Ready Static Handling:** Integrated with `WhiteNoise` for efficient, ultra-fast serving of static assets and custom UI/UX design components directly from the server.
* **Serverless Deployment:** Optimized configuration for continuous deployment via Vercel using serverless Python functions.

---

## 🛠️ Tech Stack & Tools
* **Backend:** Python, Django 6.x
* **Frontend:** HTML5, Modern CSS3 (CSS Grid & Flexbox), Semantic JavaScript
* **Deployment & Hosting:** Vercel (Serverless Functions)
* **Static Assets:** WhiteNoise
